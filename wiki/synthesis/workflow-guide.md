# 工作流指南 / Workflow Guide

## 典型工作流 / Typical Workflows

### 1. NEP势函数训练 / NEP Potential Training

```
┌─────────────────────────────────────────────────────────┐
│                     准备阶段                              │
├─────────────────────────────────────────────────────────┤
│ 1. 收集DFT训练数据                                       │
│    └─ VASP, Quantum ESPRESSO输出                         │
│                                                         │
│ 2. 转换为扩展XYZ格式                                     │
│    └─ 包含能量、力、维里应力                              │
│                                                         │
│ 3. 创建train.xyz和test.xyz                              │
│    └─ 训练集/测试集分离                                  │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                   训练阶段                                │
├─────────────────────────────────────────────────────────┤
│ 1. 编写nep.in                                           │
│    └─ 定义模型参数                                       │
│                                                         │
│ 2. 运行gen.xyz.xyz (如有需要)                           │
│    └─ 生成训练配置                                       │
│                                                         │
│ 3. 运行nep                                              │
│    └─ 训练NEP模型                                        │
│                                                         │
│ 4. 检查训练损失                                         │
│    └─ 能量、力、维里损失                                 │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                  验证阶段                                 │
├─────────────────────────────────────────────────────────┤
│ 1. 在测试集上评估                                        │
│    └─ nep_inference                                     │
│                                                         │
│ 2. 比较DFT vs NEP                                       │
│    └─ 能量、力误差                                       │
│                                                         │
│ 3. 生成nep.txt                                          │
│    └─ 最终模型文件                                      │
└─────────────────────────────────────────────────────────┘
```

### 2. MD模拟 / MD Simulation

```
┌─────────────────────────────────────────────────────────┐
│              准备阶段                                    │
├─────────────────────────────────────────────────────────┤
│ 1. 准备结构文件                                         │
│    └─ XYZ格式                                           │
│                                                         │
│ 2. 获取势函数                                           │
│    └─ nep.txt或其他                                    │
│                                                         │
│ 3. 编写run.in                                           │
│    └─ 使用LSP验证语法                                   │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│            平衡阶段                                       │
├─────────────────────────────────────────────────────────┤
│ # Equilibration run.in                                  │
│ potential nep.txt                                       │
│ velocity 300 12345                                      │
│ time_step 1                                            │
│ ensemble nvt_ber 300 300 1.0                            │
│ dump_thermo 100                                        │
│ run 100000                                             │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│            生产运行                                       │
├─────────────────────────────────────────────────────────┤
│ # Production run.in                                     │
│ potential nep.txt                                       │
│ time_step 1                                            │
│ ensemble nve                                           │
│ compute_hac 10 100 2000                                │
│ dump_thermo 100                                        │
│ dump_exyz 1000 1                                       │
│ run 1000000                                            │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│            分析阶段                                       │
├─────────────────────────────────────────────────────────┤
│ 1. 检查热力学输出                                        │
│    └─ thermo.out                                        │
│                                                         │
│ 2. 分析轨迹                                             │
│    └─ ovito, VMD                                       │
│                                                         │
│ 3. 计算性质                                             │
│    └─ hac.out → 热导率                                  │
└─────────────────────────────────────────────────────────┘
```

```
│    ensemble nve                                         │
│    run 200000                                           │
│                                                         │

│    ensemble nve                                         │

│                                                         │
│ 3. 后处理                                               │
│    └─ hnemd.out → 热导率                                 │

```

## LSP辅助工作流 / LSP-Assisted Workflow

### 开发循环
```
1. 编写输入文件
   └─ 使用编辑器LSP功能

2. 实时验证
   ├─ 语法检查
   ├─ 参数提示
   └─ 悬浮文档

3. 运行前检查
   └─ gpumd-lint ./case --json

4. 执行模拟
   └─ GPUMD

5. 错误诊断
   └─ gpumd-lint parses run.out

6. 修复和重试
   └─ 查看诊断建议
```

### MatMaster自动化
```
1. 输入生成
   └─ 工作流引擎生成run.in

2. LSP验证
   └─ gpumd-lint pre-flight check

3. 模拟执行
   └─ GPUMD运行

4. 结果验证
   └─ LSP解析运行日志

5. 数据处理
   └─ 后处理管道
```

## 常见问题排查 / Troubleshooting

### 模拟不启动
```
检查清单:
□ GPUMD010: potential是第一条命令吗?
□ GPUMD-E063: nep.txt存在吗?
□ GPUMD-W060: thermostat拼写正确吗?
```

### 热导率不合理
```
检查清单:
□ 系统充分平衡了吗?
□ 运行时间足够长吗?
□ compute_hac参数合适吗?
□ 截断半径足够大吗?
```

### NEP训练失败
```
检查清单:
□ GPUMD-E064: train_file存在吗?
□ cutoff合理吗?
□ generation足够大吗?
```
